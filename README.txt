Туториал для запуска:

    1. Написать в терминал команду git clone git@github.com:daniilcehmestrenko/test-LeadHit.git

    2. Перейти в дирректорию проекта cd test-LeadHit

    3. Создать вирутальное окружение python3.<версия вашего> -m venv env и запустить его

    4. Написать в терминал команду  chmod +x start.sh & ./start.sh

    5. Сервер запущен, для теста этого апи написан скрипт test_script.py
